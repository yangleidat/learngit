第一次搭建自己的博客，学习了hexo的安装，git的一些基本命令

### hexo的安装

#### 安装的前提

- 需要安装[Node.js](https://nodejs.org/en/)
- 需要安装[Git](github.com)

#### 安装Hexo

可以在Git Bash中使用命令安装

``` bash
$ npm install -g hexo-cli
```
